# Demo video script — target 2:45

## 0:00–0:18 — The problem

**Screen:** Classroom Mirror hero and sample lesson.

**Voiceover:** “Teachers usually discover a lesson’s hidden ambiguity only after students meet it. Classroom Mirror is a flight simulator for lessons: it stress-tests the learning path before the class begins.”

## 0:18–0:42 — Submit the lesson

**Screen:** Briefly scroll the seasons lesson and click **Mirror this lesson** in live GPT-5.6 mode.

**Voiceover:** “This lesson looks engaging, but it contains a subtle risk. The GPT-5.6 root analyzes conceptual clarity while three hosted subagents review academic language, attention and feedback, and transfer in parallel.”

## 0:42–1:28 — The evidence reveal

**Screen:** Reveal the evidence ledger. Zoom into the distance finding, exact quote, skeptic challenge, and qualified verdict.

**Voiceover:** “The model cannot simply make a claim. It must quote the exact lesson evidence. Then a skeptic challenges the finding. Here, the flashlight model may imply that distance causes seasons—but the skeptic correctly notes that the model itself can work if distance is explicitly held constant. The result is qualified, not overstated. Application code rejects any quote that does not literally occur in the lesson.”

## 1:28–1:58 — Teacher control

**Screen:** Accept two recommendations, skip one, open the redesigned sequence.

**Voiceover:** “Classroom Mirror never silently rewrites the teacher’s work. The teacher accepts or skips each intervention, then sees a revised sequence that preserves the original learning goal.”

## 1:58–2:20 — Proof

**Screen:** Show benchmark folder and a saved live results summary.

**Voiceover:** “I built a reproducible benchmark containing six lessons and twelve deliberately planted instructional failures. In the completed live GPT-5.6 run, Classroom Mirror found all twelve, and every one of its twenty-eight returned findings used an exact lesson quote. The cases and results ship in the repository.”

## 2:20–2:44 — Codex story

**Screen:** Show repository structure, tests passing, and optionally a short Codex task view.

**Voiceover:** “I built Classroom Mirror with Codex from an empty repository. Codex helped verify the new GPT-5.6 Multi-agent API, implement the product and evaluation harness, and—most importantly—challenge the original persona-based prototype. That critique led to the evidence ledger, skeptic verification, and teacher decision layer you see here.”

## 2:44–2:55 — Close

**Screen:** Product hero, then exported analysis.

**Voiceover:** “Classroom Mirror helps teachers find the barrier before a learner does. AI surfaces possibilities. Teachers make the decisions.”

## Recording checklist

- Keep the final upload under three minutes.
- Use a public YouTube URL.
- Include audible narration.
- Say both “Codex” and “GPT-5.6.”
- Do not expose the API key, terminal environment, or private Codex content.
- Use a live analysis that has already been smoke-tested before recording.
